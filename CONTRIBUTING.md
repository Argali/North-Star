# Contributing to North Star

North Star is an architecture project before it is a software project.

Contributions should improve:

- clarity
- traceability
- institutional knowledge
- human oversight

Before submitting a pull request:

1. Review the Foundation documents.
2. Ensure terminology matches the Glossary.
3. Avoid introducing concepts that duplicate existing entities.
4. Prefer extending existing models over creating new ones.

## Documentation First

Architectural changes should be documented before implementation.

## Design Principle

Complexity should be organized, not loaded.
