# North Star

## Vision

Build AI organizations that accumulate institutional knowledge without accumulating context.

## Core Principles

1. Small active context
2. Reports are first-class artifacts
3. Knowledge is extracted, not accumulated
4. Decisions are tracked separately
5. Memory is curated, not hoarded
6. Retrieval beats context loading
7. Human-readable by default

## Success Criteria

A system with 10,000 reports should remain as usable as one with 10 reports.
